# 🎮 Counter Guide (CSG)

> Guias rápidos de utilitários para Counter-Strike 2.
[![Status](https://img.shields.io/badge/status-active-success.svg)]()

---
## 📋 Sobre o Projeto

O **Counter Guide** é uma plataforma web focada em guias rápidos e práticos de utilitários (smokes, flashbangs, etc.) para **Counter-Strike 2**. 

O projeto foi desenvolvido com o objetivo de ajudar jogadores a aprenderem nades de forma rápida e organizada.

### 🎯 Funcionalidades Principais
- Seleção de mapas
- Abas separadas por Smoke e Flashbang
- Tutoriais por lado (CT / TR)
- Vídeos integrados do YouTube
- Design neon moderno

---
## 🚀 Tecnologias Utilizadas

### Backend:
- **Node.js** + **Express.js**
- **SQLite** (banco de dados)

### Frontend:
- **HTML5** + **CSS3** (com estilo neon)
- **JavaScript** (interatividade)

### Outros:
- API RESTful
- Integração com YouTube
